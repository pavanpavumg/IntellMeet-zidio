# 🚀 IntellMeet – AI-Powered Meeting Platform

📌 Built during **Zidio Development Internship (March 2026)**

An enterprise-grade meeting and collaboration platform with real-time video, AI-powered summaries, and team collaboration features.

---

## 🔥 Features
- 🎥 Real-time Video Meetings (WebRTC)
- 💬 Live Chat (Socket.io)
- 🤖 AI Meeting Summary & Action Items
- 📊 Dashboard & Analytics
- 🔐 Secure Authentication (JWT)

---

## 🛠 Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- Real-time: Socket.io, WebRTC
- AI: OpenAI API
